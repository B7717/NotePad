## 项目简介
### 项目名称：Android简易记事本APP
### 项目功能：新建笔记、编辑笔记、删除笔记、搜索笔记等
### 项目语言：Java
### 项目平台：Android Studio
### 项目作者：121052022055 软工二班 张灿然


## 项目功能
### 添加时间戳
#### 功能说明：当用户新建笔记时，系统会自动添加当前时间戳
